```markdown
# HelloWorld gRPC Server Application

## Обзор

Это простейшее серверное приложение, которое использует gRPC для вызова его клиентами и передачи простого приветственного собщения. 

## gRPC Сервер

Клиентом может быть условный постман или отдельное клиентское приложение:

[https://github.com/ArseniyZhivyh/gRPChelloWorld/tree/master]

Перед запуском клиентского приложения убедитесь, что сервер запущен и доступен.

## Запуск сервера

Клонируйте себе репозиторий и запустите App.main()

```
